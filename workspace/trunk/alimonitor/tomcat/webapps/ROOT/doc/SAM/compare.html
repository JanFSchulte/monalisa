The values from the 4 Patricia's scripts are now sent to the local ML on each of the 3 voboxes as

{CERN-L|CNAF|FZK}/SAM_SiteJobs/{batch|is|lb|gris}/{running|waiting}.

If you want to send also other values, you can use the small script I have written for this, that allows you to send directly from shell various parameters to a ML service. By default, it will send to localhost. If you define MONALISA_HOST with a hostname, it will send them there. Just call

sendToML.sh <cluster> <node> <param1> <val1> [ <param2> <val2> .. ]

I've copied this together with a perl ApMon installation on the 3 VoBoxes in ~/SAM/sendToML, or you can get it from:

http://monalisa2.cern.ch/~catac/apmon/sendToML.tgz
